# Python Programs Collection

This repository contains a collection of Python programs created for practice, learning, and problem-solving. It includes basic to intermediate level programs covering different concepts of Python programming.

## About

This project is designed to:

* Practice Python programming
* Improve problem-solving skills
* Understand core programming concepts
* Prepare for coding interviews and academic tasks

## Project Structure

The repository includes multiple Python files such as:

* Basic programs (input/output, operators)
* Conditional statements (if-else)
* Loops (for, while)
* Functions
* String manipulation
* List & array operations
* Mathematical problems
* Logic building exercises

Each file contains a separate program solving a specific problem.

## How to Run

1. Install Python (if not installed)
2. Clone the repository:

   ```bash
   git clone https://github.com/RajVadaviya/python_program.git
   ```
3. Navigate to the folder:

   ```bash
   cd python_program
   ```
4. Run any file:

   ```bash
   python filename.py
   ```
